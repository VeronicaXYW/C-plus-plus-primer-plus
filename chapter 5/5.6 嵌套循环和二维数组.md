二维数组的创建、使用规则
```
int max[4] [5] = 
//在电脑中二维数组依次存储，01、02、03...05、21...25、31...35
{
	{ 此处放第一行的5个数字 },//一行视为一维数组中的一个元素，所以相互之间用逗号分隔
	{ 第二行的5个数字 },
	...
};//最后用分号
```
